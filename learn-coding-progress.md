## Notes

**Tech architecture layers for web solutions:**

| Layer | Responsibility |
|---|---|
| Access Layer | UI, API, Authentication |
| Application Layer | Orchestration of services |
| Domain Layer | Core business logic |
| AI Processing Layer | AI calls and inference handling |
| Data Layer | Databases, caching |
| Communication Layer | API gateway, middleware |
| Infrastructure Layer | Cloud services, deployment |
| Security Layer | Data encryption, user auth |
