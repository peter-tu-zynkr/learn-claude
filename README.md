# Learn Claude

A personal learning journal for exploring AI-powered coding tools and modern development workflows.

## What's Inside

- **learn-coding-progress.md** — Checklist tracking progress across Claude Code, GitHub, Cursor, MCP integrations, and more
- **Term glossary.md** — Plain-language glossary of coding terms (frontend, backend, cloud, AI, databases, etc.)
- **claude-with-api.md** — Notes on using Claude Skills with the Messages API, Code Execution tool, and Files API
- **CLAUDE.md** — Project-level instructions for Claude Code

## Topics Covered

- Claude Code setup and commands
- GitHub CLI and Git workflows
- Skills, Sub Agents, and MCP server integrations
- Cursor IDE
- Frontend (JavaScript/TypeScript) and Backend (Python/FastAPI)
- Databases (SQL, PostgreSQL, Supabase)
- Cloud infrastructure (AWS, Cloudflare, Vercel, Railway)
- Deployment (Vercel, Zeabur)
